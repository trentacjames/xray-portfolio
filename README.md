# Xray Portfolio Dashboard

Xray is a portfolio exposure dashboard that gives investors a clear, real-time view of their underlying asset exposures across funds, ETFs, and other investment vehicles.

## What Xray Does

Xray "looks through" the wrapper of each holding in a portfolio — ETFs, mutual funds, managed accounts — and aggregates the true underlying exposures. This lets you see:

- **Asset class breakdown** — equities, fixed income, alternatives, cash
- **Geographic exposure** — country and regional allocation
- **Sector concentration** — where your capital is actually deployed
- **Top underlying holdings** — the real positions driving your returns

## Project Structure

```
xray-portfolio/
├── src/         # Application source code
├── public/      # Web UI (HTML, CSS, JS)
├── data/        # Portfolio data files
└── README.md
```

## Getting Started

Install dependencies and start the development server:

```bash
npm install
npm start
```
